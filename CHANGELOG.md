# Changelog

## Rolume 1.1

小范围公开测试版本。

### 下载

- 下载：[Rolume.dmg](https://github.com/ericcilcn/Rolume/releases/download/1.1/Rolume.dmg)
- SHA-256：`1321c68153f2caad5033a998ec83e91d57fe0d0419787299451b749aaee26b26`

### 公开测试修正

- 修复开启鼠标滚轮反转后，触控板滑动调节音量手感变钝的问题
- 改进触控板事件识别，避免鼠标滚轮反转逻辑接管触控板路径
- 重写应用内 FAQ，补充 DDC/CI、辅助功能权限、多显示器控制和自然滚动说明
- 更新发布说明，移除发布页上的第三方创作标记

### 品牌与项目整理

- 项目从 `MacVolumeControl` 更名为 `Rolume`
- 更新 App 名称、图标、Xcode 工程和 GitHub 仓库名称
- 统一 README、应用内 GitHub 链接和发布链接
- 清理旧版命名残留和历史安装包文件

### 外接显示器与音量控制

- 重构外接显示器 DDC/CI 音量控制逻辑
- 增强外接显示器识别和音频设备匹配
- 改进显示器拔插后的音量控制恢复能力
- 修复 DDC 最大值异常导致音量溢出的问题
- 修复 OSD、设置页滑条和实际音量不同步的问题
- 修复菜单栏音量图标不随音量变化更新的问题

### 鼠标、触控板与滚动逻辑

- 鼠标和触控板改为独立音量控制设置
- 支持鼠标和触控板分别设置步进幅度
- 支持鼠标和触控板分别设置修饰键
- 支持独立配置滚动和滑动拦截
- 修复鼠标滚轮方向在翻转设置下与 OSD 不一致的问题
- 修复拔插显示器后滚轮方向偶发反向的问题
- 优化全局滚动事件拦截逻辑
- 优化热路径设置缓存，减少滚动时的卡顿感

### 权限与系统体验

- 改进辅助功能权限请求流程
- 使用 macOS 原生权限提示替代自定义打开设置弹窗
- 授权成功后显示 Rolume 专属重新打开提示
- 修复语言设置为跟随系统时错误显示英文的问题

### 设置界面与 OSD

- 优化设置页标签布局，减少图标和文字漂移
- 调整设置窗口尺寸，避免内容切换时跳动
- 重排关于页布局
- 优化支持 Rolume、GitHub、FAQ 和联系方式入口
- 调整 OSD 位置，使其更接近 macOS 原生音量提示
- 调整 OSD 尺寸、圆角和设备名布局
- 保留稳定的 `.hudWindow` OSD 呈现方式

### 安装包与发布

- 重做 DMG 安装包
- 添加 DMG 背景图
- 固定 DMG 窗口大小和图标位置
- 提供拖拽到 Applications 的安装体验
- 发布 `Rolume 1.1` GitHub Release
- 上传 `Rolume.dmg`
- 补充 SHA-256 校验值
- 补充 README 下载链接、权限说明和支持项目链接

### 说明

- Rolume 需要 macOS 14.0 或更高版本
- 只有启用鼠标滚轮反转、拦截滚动或拦截滑动时才需要辅助功能权限
- 当前公开测试版尚未使用 Developer ID 签名和 notarization
